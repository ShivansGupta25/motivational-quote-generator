# 📜 Motivational Quote Generator

A simple and responsive Motivational Quote Generator built using **HTML, CSS, and Vanilla JavaScript**.  
Click the button to generate a random inspirational quote with its author.

---

## 🚀 Live Demo

https://shivansgupta25.github.io/motivational-quote-generator/

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

## ✨ Features

- 🎲 Generates random motivational quotes  
- 👤 Displays quote with author name  
- 🎨 Clean and modern UI  
- 📱 Responsive design

## 🧠 How It Works

- Quotes are stored in a JavaScript array.
- When the button is clicked:
  - A random index is generated.
  - The quote and author are displayed dynamically on the page.
